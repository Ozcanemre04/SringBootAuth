# SringBootAuth

  ## Description
  I did a basic jwt authentication and crud operation to learn spring boot


  ## How to setup the project
  1. git clone 'Project url'
  2. create .env
  3. add in .env :
     - DB_USERNAME= {your db username}
     - DB_PASSWORD= {your password}
     - DB_URL=jdbc:postgresql://{host}:{port}/{your db name}
     - JWT_SECRET={your token}
     - JWT_EXPIRATION={expiration date of your token}
